# Installation info
- Run the following command to clean-install
```sh
git clone https://github.com/maniac-en/dotfiles.git "$HOME/"dotfiles \
	&& cd "$HOME"/dotfiles && chmod a+x install.sh \
	&& sudo ./install.sh
```
